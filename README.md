Repository ini merupakan Dummy Proyek saya mengenai Analisis wilayah rawan gempa di Jawa Barat

Dataset diambil dari kaggle () yang berupa dataset kejadian gempa di indonesia berjumlah 131.833 data. Setelah di filter menjadi bagian Jawa Barat jumlah Dataset menjadi 1.256 data.
Untuk membagi wilayah Jawa Barat per Kabupaten menggunakan GeoPandas dengan data geoBoundaries-IDN-ADM2. Lalu model yang digunakan untuk clustering wilayahnya menggunakan KMeans.

Setelah di analisis, wilayah yang paling banyak terjadi kasus gempa yaitu di Kab. Cianjur dengan jumlah kasus sekitar 580 kasus gempa, Kab. Sukabumi dengan jumlah kasus sekitar 230 kasus gempa. Dan wilayah rawan di Jawa Barat dapat di kategorikan menjadi tiga yaitu rawan rendah, rawan sedang dan rawan tinggi
